## 🔍 Sample Case Analysis

### 🚨 Case: Brute Force Attack

**Scenario:**  
Multiple failed login attempts followed by a successful login from an external IP.

**Findings:**
- Event ID 4625 → Failed logins  
- Event ID 4624 → Successful login  
- Logon Type 10 → Remote access  
- Source IP → External & suspicious  

**Analysis:**  
The attacker attempted multiple credentials (brute force attack) and eventually gained access to a valid account.

**Conclusion:**  
🔴 Confirmed malicious activity (Account Compromise)
